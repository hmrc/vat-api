post

Submit VAT return for period.

post /vat/{vrn}/periods/{periodKey}

Submit VAT return for period.

  • Request
  • Response

URI Parameters

  • vrn: required (string)

    Vat registration number.

    Example:

    123456789
  • periodKey: required (string)

    Identifier for the period of the VAT return

    Example:

    1234567
